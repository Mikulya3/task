Пункт 11: Реклама должна отображаться каждое утро и вечер в будние дни.
В каком часовом поясе? Указаны только будние дни, а как насчет выходных?
4. Пункт 5 кажется бессмысленным.
5. Пункт 6: какой лимит товаров превышен или общий?

Пользователь может добавить от 1 до 10 единиц одного товара.
      1. В корзине может быть не более 5 разных товаров.
      2. Общее количество всех товаров в корзине не может превышать 20 единиц.
      3. Если добавление товара приводит к превышению любого лимита, система показывает конкретное сообщение об ошибке, указывающее, какой лимит превышен.
      4. Пользователь может изменить количество товара на любое значение от 1 до 10, если при этом соблюдаются общие лимиты.
      5. Товар можно удалить из корзины либо кнопкой удаления, либо установкой количества в 0.
      6. Цена товара фиксируется в момент добавления в корзину и впоследствии не изменяется.
      7. На странице корзины отображаются:
      список товаров;
      количество;
      цена за единицу;
      общая цена за каждый товар;
      итоговая сумма корзины.
      8. Корзина может показывать рекламные баннеры, которые не влияют на содержимое корзины.
      9. Реклама показывается с понедельника по пятницу с 08:00 до 10:00 и с 18:00 до 20:00 (местное время).
      10. Изменения цен в каталоге не влияют на уже существующие корзины, а затрагивают только новые добавления.
Вопросы к менеджеру.
    1. Какое правило по цене верное: фиксированная цена или динамическое обновление?
    2. Что именно считается лимитом корзины — нужно ли показывать отдельные ошибки для каждого правила?
    3. Снижение количества до 0 — основной способ удаления товара или кнопка удаления обязательна?
    4. Как именно должна отображаться реклама: баннер, карточка товара, всплывающее окно?
    5. Какой точно часовой пояс и временные интервалы для показа рекламы?
    6. Лимиты применяются по сессии или по учётной записи пользователя?
    7. Должна ли корзина сохраняться после выхода из аккаунта?
